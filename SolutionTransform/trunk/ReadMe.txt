Usage:

SolutionTransform <Script Full Path> <Solution Full Path>

e.g.
SolutionTransform C:\Projects\scratch\Reference35\SolutionTransform\Scripts\CastleSilverlight.boo C:\OSS3\Castle\Castle-all-vs2008.sln

This creates a silverlight version of the original trunk project (at least on my machine)

e.g.

SolutionTransform C:\Projects\scratch\Reference35\SolutionTransform\Scripts\CastleStandardize.boo C:\Projects\scratch\Reference35\SolutionTransform\SolutionTransform.sln

This was used to ensure compliance with Castle coding standards when the code was originally checked in.
